## Welcome to New API Documentation

This section contains the complete API documentation for New API, covering the following:

### Relay Interfaces

- **Chat** - Support for multiple chat models including OpenAI, Anthropic, Google Gemini
- **Embeddings** - Text vectorization interfaces
- **Rerank** - Document reranking interfaces
- **Realtime** - WebSocket real-time chat interfaces
- **Image** - AI image generation interfaces
- **Audio** - Speech-to-text and text-to-speech interfaces
- **Music** - AI music generation interfaces
- **Video** - AI video generation interfaces

### Query Interfaces

- **Token Usage** - Query and statistics for token usage

### Frontend Interfaces

- **Authentication System** - User authentication and authorization
- **Available Models List** - Get all supported models
- **API Modules** - Complete backend management interfaces

<Callout type="info">
  All interface prefixes are unified as `http(s)://<your-domain>`. Please use HTTPS in production environments for security.
</Callout>
