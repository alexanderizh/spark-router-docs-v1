<Callout type="info" title="Chat Settings Options">
  In the System Settings -> Chat Settings of the Spark Router console, the following
  shortcut options can be added to easily populate Cherry Studio with a single
  click on the token management page:
</Callout>

    ```json
    { "Cherry Studio": "cherrystudio://providers/api-keys?v=1&data={cherryConfig}" }
    ```

<Callout type="info">
  🍒 Cherry Studio is a powerful desktop AI client designed for professional
  users, integrating 30+ industry-specific intelligent assistants to meet the
  needs of various work scenarios and significantly improve work efficiency.
</Callout>

## Spark Router Integration Method

### Parameter Configuration

Provider Type: Spark Router supported types
API Key: Obtained from Spark Router
API Address: Spark Router site address
