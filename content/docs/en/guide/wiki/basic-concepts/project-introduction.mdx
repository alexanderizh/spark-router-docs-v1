### Core Features

- **Unified Interface**: One API endpoint for all AI services, OpenAI-compatible format
- **Intelligent Routing**: Multi-channel load balancing, automatic failover, weighted distribution
- **Flexible Billing**: Per-request/usage-based pricing, prepaid top-up, configurable rate multipliers
- **Security Control**: Token permission management, model access control, API call auditing
- **Data Insights**: Real-time dashboard, usage statistics, cost analytics
- **Multi-Tenant Architecture**: Perfect for individual developers, team collaboration, and enterprise deployment
