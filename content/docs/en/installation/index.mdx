## Installation & Deployment Guide

This section provides various installation and deployment solutions for New API, along with system configuration instructions.

### Installation Methods

We offer multiple flexible deployment options. Choose the one that best fits your needs:

#### Recommended Methods

- **Docker Compose Installation** - The simplest and fastest deployment method, suitable for most users
- **Docker Installation** - Deploy using Docker containers

#### One-Click Deployment

- **1Panel Installation** - One-click installation via 1Panel control panel
- **BT Panel Docker Installation** - Deploy using Docker in BT Panel

#### Advanced Deployment

- **Cluster Deployment** - High-availability cluster deployment solution
- **Local Development** - Build from source and set up development environment

### Configuration & Maintenance

- **System Update** - How to update to the latest version
- **Environment Variables** - Environment variable configuration guide
- **Docker Compose Configuration** - Detailed Docker Compose file explanation

<Callout type="info">
  For production environments, we recommend using Docker Compose or cluster
  deployment with HTTPS certificates configured for security.
</Callout>

<Callout type="warn">
  Before deployment, please ensure your server meets the minimum requirements: -
  CPU: 1 core or more - Memory: 512MB or more (2GB recommended) - Storage: 10GB
  or more available space
</Callout>
