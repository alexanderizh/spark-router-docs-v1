## 部署安装指南

本节提供 New API 的各种安装部署方案和系统配置说明。

### 安装部署方式

我们提供了多种灵活的部署方式，您可以根据实际需求选择：

#### 推荐方式

- **Docker Compose 安装** - 最简单快捷的部署方式，适合大多数用户
- **Docker 安装** - 使用 Docker 容器化部署

#### 一键部署

- **1Panel 安装** - 通过 1Panel 控制面板一键安装
- **宝塔 Docker 安装** - 在宝塔面板中使用 Docker 部署

#### 高级部署

- **集群部署** - 高可用集群部署方案
- **本地开发** - 从源码编译和开发环境搭建

### 配置与维护

- **系统更新** - 如何更新到最新版本
- **环境变量** - 环境变量配置说明
- **Docker Compose 配置** - Docker Compose 文件详细说明

<Callout type="info">
  建议生产环境使用 Docker Compose 或集群部署方式，并配置 HTTPS
  证书以确保安全性。
</Callout>

<Callout type="warn">
  在开始部署前，请确保您的服务器满足最低配置要求： - CPU: 1核心以上 - 内存:
  512MB 以上（推荐 2GB） - 硬盘: 10GB 以上可用空间
</Callout>
