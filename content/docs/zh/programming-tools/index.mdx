## 概述

本栏目汇总在 Claude Code、Cursor 等编程工具中使用 Spark Router（Claude API）的配置方法与实践建议，便于开发者统一管理 API 并提升使用体验。

## 文档列表

- **[Claude Code](/zh/docs/programming-tools/claude-code)**：在 Claude Code 中配置 Spark Router，含配置文件、cc-switch 等供应商配置方式及最佳实践。
- **[Cursor](/zh/docs/programming-tools/cursor)**：在 Cursor 中配置 Spark Router，含 Base URL 设置、注意事项及最佳实践。
- **[其他工具](/zh/docs/programming-tools/other-tools)**：支持 OpenAI 标准接口的应用均可按相同方式接入，说明通用配置要点。
