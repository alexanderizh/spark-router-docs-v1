## 配置步骤

Cursor 环境遵循 OpenAI 接口标准进行配置，具体流程如下：

![Cursor 添加模型](/assets/programming-tools/2.png)

1. 打开 Cursor 设置界面
2. 导航至【Models】模型管理部分
3. 选择【添加模型】功能
4. 选择【添加 OpenAI API Key】选项
5. 在【OpenAI Base URL】字段中配置基础地址

## 重要注意事项

<Callout type="warn" title="Base URL 配置规范">
  **Base URL 配置规范：**
  - 必须在基础 URL 末尾添加 `/v1` 后缀
  - 示例：`http://sparkrouter.ai/v1`
  - 这是确保与 OpenAI 接口标准兼容的关键步骤
</Callout>

## 最佳实践建议

### 安全性建议

- API 令牌应妥善保管，避免在公开场合展示完整令牌
- 建议使用环境变量管理敏感配置信息
- 定期轮换 API 密钥以提高安全性

### 性能优化

- 根据实际需求选择合适的模型版本
- 合理设置请求超时和重试机制
- 监控 API 使用情况，优化调用频率

### 故障排除

1. **连接问题**：检查 Base URL 是否正确（含 `/v1` 后缀），网络是否通畅
2. **认证失败**：验证 API 令牌是否有效
