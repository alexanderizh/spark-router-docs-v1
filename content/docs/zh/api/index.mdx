## 欢迎使用 New API 接口文档

本节包含了 New API 的完整接口文档，涵盖以下内容：

### 中继接口

- **对话接口** - 支持 OpenAI、Anthropic、Google Gemini 等多种对话模型
- **嵌入向量** - 文本向量化接口
- **重排序** - 文档重排序接口
- **实时通信** - WebSocket 实时对话接口
- **图像生成** - AI 图像生成接口
- **音频处理** - 语音转文字、文字转语音接口
- **音乐生成** - AI 音乐创作接口
- **视频生成** - AI 视频创作接口

### 查询接口

- **Token 使用统计** - 查询和统计 Token 使用情况

### 前端接口

- **认证系统** - 用户认证和授权说明
- **可用模型列表** - 获取系统支持的所有模型
- **API 管理模块** - 完整的后台管理接口

<Callout type="info">
  所有接口前缀统一为 `http(s)://<your-domain>`，生产环境请使用 HTTPS 以保证安全性。
</Callout>
