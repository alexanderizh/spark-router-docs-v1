## インストール・デプロイガイド

このセクションでは、New API のさまざまなインストールとデプロイソリューション、およびシステム設定の説明を提供します。

### インストール方法

お客様のニーズに最適な複数の柔軟なデプロイオプションを提供しています：

#### 推奨方法

- **Docker Compose インストール** - 最もシンプルで迅速なデプロイ方法、ほとんどのユーザーに適しています
- **Docker インストール** - Docker コンテナを使用したデプロイ

#### ワンクリックデプロイ

- **1Panel インストール** - 1Panel コントロールパネル経由のワンクリックインストール
- **BT パネル Docker インストール** - BT パネルで Docker を使用したデプロイ

#### 高度なデプロイ

- **クラスターデプロイ** - 高可用性クラスターデプロイソリューション
- **ローカル開発** - ソースからのビルドと開発環境のセットアップ

### 設定とメンテナンス

- **システム更新** - 最新バージョンへの更新方法
- **環境変数** - 環境変数の設定ガイド
- **Docker Compose 設定** - Docker Compose ファイルの詳細な説明

<Callout type="info">
  本番環境では、セキュリティのために HTTPS 証明書を設定した Docker Compose
  またはクラスターデプロイの使用を推奨します。
</Callout>

<Callout type="warn">
  デプロイ前に、サーバーが最小要件を満たしていることを確認してください： - CPU:
  1コア以上 - メモリ: 512MB以上（2GB推奨） - ストレージ: 10GB以上の空き容量
</Callout>
