## New API ドキュメントへようこそ

このセクションには、New API の完全な API ドキュメントが含まれており、以下の内容をカバーしています：

### リレーインターフェース

- **チャット** - OpenAI、Anthropic、Google Gemini などの複数のチャットモデルをサポート
- **エンベディング** - テキストベクトル化インターフェース
- **リランク** - ドキュメント再ランキングインターフェース
- **リアルタイム** - WebSocket リアルタイムチャットインターフェース
- **画像** - AI 画像生成インターフェース
- **音声** - 音声からテキスト、テキストから音声へのインターフェース
- **音楽** - AI 音楽生成インターフェース
- **動画** - AI 動画生成インターフェース

### クエリインターフェース

- **トークン使用統計** - トークン使用状況のクエリと統計

### フロントエンドインターフェース

- **認証システム** - ユーザー認証と承認
- **利用可能なモデルリスト** - サポートされているすべてのモデルを取得
- **API モジュール** - 完全なバックエンド管理インターフェース

<Callout type="info">
  すべてのインターフェースのプレフィックスは `http(s)://<your-domain>` に統一されています。本番環境ではセキュリティのため HTTPS を使用してください。
</Callout>
